Block-Pursuit

  • Java
  • Python
  • JavaFX
  • Javascript

Block-Pursuit is 2d survival game. How long will you be able to avoid the AI as both your speeds increase? Includes multiple modes and maps and a file system for tracking high scores. AI uses an unique/original path finding algorithm, which I call the lillypad method. You can toggle visuals for the lillypad method.
